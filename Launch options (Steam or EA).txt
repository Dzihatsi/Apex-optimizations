Основные параметры
+exec autoexec.cfg            // Используется для загрузки пользовательской конфигурации. Сам конфиг мы настрое чуть позже
+fps_max 180                  // Ограничивает Fps до нужного вам значения. Лучше не использовать сторонние программы для ограничения Fps во избежание задержек ввода. При указании значения 0 Fps не будет ограничен
-preload                      // Предварительная загрузка текстур и объектов
-forcenovsync                 // Отключает вертикальную синхронизацию
-high                         // Задает высокий приоритет для игры. Все выделенные ресурсы будут
-threads 8                    // Задает количество потоков процессора
+gfx_nvnUseLowLatency 1       // Включает NVIDIA Reflex (Снижает системную задержку и повышает отклик)
-freq 165                     // Задает герцовку, если игра не может ее определить 
-dxlevel95                    // Запуск через DirectX9 (Из своего опыта могу сказать, даже если вы обладатель видеокарты которая поддерживает DX11-12, запуск через DX95 снизит нагрузку на Gpu и повысит производительность) 
                                 При этом игра останется все так же стабильной по умолчанию вы запускаетесь через DX11, который менее производительный чем DX95, но более надежный и стабильный чем DX12

Дополнительные параметры
+m_rawimput 1                 // Прямое подключение мыши без учета настроек ОС
-dev / -novid / -nomenuvid    // Отключают вступительное видео при входе в игру
-fullscreen                   // Запуск игры в полноэкранном режиме
+cl_showpos 1                 // Включает показатели скорости, угла и положения персонажа
+cl_showfps 4                 // Включает счетчик Fps в игре
+cl_is_softened_locale 1      // Уменьшает визуальный шум, заменяя брызги крови при нокауте или урону по HP на менее яркую красную вспышку
